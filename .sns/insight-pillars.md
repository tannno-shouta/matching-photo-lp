# Insight Pillars — 福岡垢抜けマッチングフォト 発信の5本柱

「**垢抜けで、出会いの入口を開ける**」を最終提供価値とし、5軸で発信する。
すべての投稿はいずれかの柱に紐づく。柱を跨ぐ投稿はOK、柱を持たない一般論はNG。

## 三層構造（全柱の背骨）

```
[表層] 垢抜け（服・髪・眉・表情）  ← 入口を整える（柱1〜3）
   ↓
[中層] 写真で世界観を作る         ← ロケ・空気感・自然な表情（柱2）
   ↓
[深層] マッチング後の関係性       ← 自信・ライフスタイル・継続的な変化（柱5）
```

柱1〜4 は **表層・中層**（写真と垢抜けの直接効果）を語る。
柱5 は **深層**（マッチング後の関係性・自分のブランド）を語る。
週次の発信比率の目安：表層・中層 7〜8割：深層 2〜3割。

## 共通スタンス（全柱共通）

**順序は常に「比喩 → 現実への橋渡し → 客観メリット → 余韻（体言止め）」**。

1. **比喩**: 家・服・店構え・名刺など、読み手が頷ける入口
2. **橋渡し**: 「マッチングアプリも、同じ」「写真も、同じ」
3. **客観メリット**: いいね↑ / マッチ率↑ / 返信率↑ / 初回デート率↑ から1〜3個
4. **余韻**: 体言止めで断言（「〜だ」「〜だよ」を抜く）

**書き出しNG**: 「中身が大事」「努力が大事」式の精神論
**書き出しOK**: 比喩・観察された現象・ペインの直接言語化・データへの言及

## 柱 1: 写真を変えると、いいね数とマッチ率が変わる

**読者ペイン**: 「いいねが0〜数件で止まっている」「マッチしない原因が自分でも分からない」

**主張**: マッチングアプリの判断は、プロフィールを開く前の**一覧画面**でほぼ終わっている。だから写真を変えるだけで、いいね数・マッチ率は変わる。中身を磨く前に、まず写真。

**比喩素材**:
- 同じ家でも、外見が整った物件と、ボロボロの物件があったら前者を選ぶ
- 飲食店も、店構えで入るかどうか決める
- 名刺が安っぽい人とは、最初の話がかみ合いにくい

**客観メリット辞書**:
- いいね数↑：一覧画面で止まる確率が上がる
- マッチ率↑：マッチング成立の確率が上がる
- 返信率↑：メッセージのやり取りが続く確率が上がる
- 初回デート率↑：会う前段階の壁が下がる

**書き出し例**:
- 「マッチングアプリは、写真が9割。」
- 「いいねが0で止まってる人ほど、中身を磨こうとする。」
- 「中身を見てもらえるのは、入口を抜けた人だけ。」

## 柱 2: 写真の "世界観" がマッチング相手の質を決める

**読者ペイン**: 「マッチはするけど、来る人のタイプが違う」「会いたい層に届かない」

**主張**: 写真は単に「映り」だけじゃなく、**ロケーション・服装・空気感**で世界観を作っている。その世界観が、自分が会いたい層を引き寄せる。雑な写真は、雑な層を引き寄せる。

**比喩素材**:
- ホテルラウンジの写真と、コンビニ前の自撮り。同じ人でも、来る相手は違う
- 高級時計と量販店の腕時計。価格より、纏ってる空気感が判断を決める
- カフェも、内装で客層が決まる。HPも、写真も、同じ構造

**客観メリット辞書**:
- マッチ層の質↑：会いたい層が反応しやすくなる
- ミスマッチ↓：不要なやりとりが減る
- 第一印象↑：「丁寧に生きてる人」感が伝わる

**書き出し例**:
- 「同じ人でも、ホテルラウンジで撮るか、コンビニ前で撮るかで来る相手が変わる。」
- 「写真の "世界観" で、マッチする層は決まる。」
- 「雑な写真は、雑な層しか引き寄せない。」

## 柱 3: 撮る前の "垢抜け" が、写真の上限を決める

**読者ペイン**: 「カメラマンに頼んだけど、思ったほど変わらなかった」「結局、自分の素材が問題なのでは」

**主張**: カメラマンは「**撮影技術**」を提供する。でも素材（服・髪・眉・表情）が整っていなければ、技術の天井にすぐ当たる。**撮る前の垢抜け** が写真の上限を決める。

**比喩素材**:
- 同じ料理人でも、食材が違えば結果が違う
- 高級ブランドの服も、中の体型と組み合わせで変わる
- スタジオで撮っても、寝癖と部屋着では限界がある

**客観メリット辞書**:
- 写真の上限↑：素材を整えると、技術が活きる
- 自然な表情↑：服が決まっていると、姿勢と表情が自然になる
- 自信↑：整えた跡からくる、内面の変化

**書き出し例**:
- 「同じ料理人でも、食材が違えば結果が違う。」
- 「カメラマンが提供してるのは、撮影技術。素材は別問題。」
- 「写真の上限は、撮る前の段階で決まっている。」

## 柱 4: 他のカメラマン vs 本サービスの構造的な違い

**読者ペイン**: 「マッチング写真って、どこに頼むのが正解か分からない」「友達に撮ってもらうのと、プロに頼むの、何が違う？」

**主張**: 他のカメラマンは「**撮るだけ**」、本サービスは「**垢抜け（服・髪・眉・表情）+ 撮る**」のオールインワン。同じ撮影技術でも、撮られる素材が違うから結果が違う。

**比喩素材**:
- 同じレストランでも、食材を仕入れる店と仕入れない店で味が違う
- ヘアサロンも、カットだけと、骨格診断＋カットでは仕上がりが違う
- パーソナルトレーナーも、トレーニングだけと、食事指導込みでは結果が違う

**客観メリット辞書**:
- 結果の差↑：素材＋技術の合わせ技で天井が上がる
- リスク↓：「撮ってもらったのに変わらなかった」が起きにくい
- コスパ↑：服・髪を別で整える費用と時間を一本化できる

**書き出し例**:
- 「カメラマンが撮るだけだと、素材は変わらない。」
- 「同じ料理人でも、食材を仕入れる人と、出来合いを使う人がいる。」
- 「マッチング写真サービス、全部同じに見えるけど構造が違う。」

## 柱 5: 垢抜けは、マッチング後の関係性と自信まで変える

**読者ペイン**: 「マッチして会えても、続かない」「自分に自信が持てない」「そもそも自分のスタイルが分からない」

**主張**: 垢抜けて整えた状態は、**写真の中だけで完結しない**。日常の自信、選ばれる回数、関係性の深まりまで影響する。マッチングは入口、その先のライフスタイルまで変える。

**比喩素材**:
- 美容室で髪を整えた帰り道、姿勢が変わる感覚
- 服が決まっている日と、決まっていない日で、声の出方まで違う
- 整っている人は、選ばれる回数より、選べる回数が多い

**客観メリット辞書**:
- マッチング後の関係継続↑：第一印象と実物のギャップが減る
- 自分への自信↑：整えた事実が、内面に効く
- 選択肢↑：「選ばれる」から「選ぶ」側へ

**書き出し例**:
- 「美容室で髪を整えた帰り道、姿勢が変わる感覚、わかると思う。」
- 「整っている人は、選ばれる回数より、選べる回数が多い。」
- 「垢抜けは、写真の中だけで終わらない。」
